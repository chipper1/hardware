Overview

All informations related to the Processes running on your sytem, defined by Process IDentifiers.

pid = Hardware::PID.new           # Default is Process.pid
app = Hardware::PID.new "firefox" # Take the first matching PID

loop do
  sleep 1
  pid.cpu_used      # => 1.5
  app.cpu_used.to_i # => 4
end
